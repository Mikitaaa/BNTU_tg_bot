# 🏠 Telegram Bot - Помощник по общежитию БНТУ

## Описание
Телеграм бот-помощник для студентов Белорусского национального технического университета (БНТУ). Бот предоставляет информацию об общежитии, правилах проживания, питании, спортивных объектах и помогает ответить на вопросы студентов с использованием AI.

### 🎯 Функциональность
- 📚 Предоставление информации об общежитии БНТУ
- 🎯 Интерактивное меню с удобной навигацией
- 💬 Ответы на свободные вопросы через Google Gemini AI
- 🔄 Легкая интеграция с базой данных
- 👤 Поддержка работы с пользовательскими данными

## Установка

### 1. Предварительные требования
Убедитесь, что у вас установлены:
- **Python 3.9+**
- **pip** (менеджер пакетов Python)

Для macOS (если нет Python):
```sh
brew install python3
```

### 2. Клонирование репозитория
```sh
git clone https://github.com/Mikitaaa/BNTU_tg_bot.git
cd BNTU_tg_bot
```

### 3. Создание виртуальной среды
```sh
python3 -m venv .venv
source .venv/bin/activate  # Для macOS/Linux
# или
.venv\Scripts\activate  # Для Windows
```

### 4. Установка зависимостей
```sh
pip install -r requirements.txt
```

## ⚙️ Конфигурация

### Переменные окружения
Для корректной работы бота необходимо установить переменные окружения. Создайте файл `.env` в корневой директории проекта или экспортируйте переменные в систему:

```bash
# Обязательные переменные:
export BOT_TOKEN="ваш_токен_бота_от_BotFather"
export GEMINI_API_KEY="ваш_API_ключ_Google_Gemini"

# Опциональные переменные:
export GEMINI_PROMPT="Ты помощник общежития БНТУ. Ответь кратко: {user_message}"
```

### Где получить необходимые ключи:

#### 1. **BOT_TOKEN** - токен Telegram бота
- Напишите в Telegram боту [@BotFather](https://t.me/botfather)
- Используйте команду `/newbot`
- Следуйте инструкциям и получите токен
- Пример: `123456789:ABCdefGHIjklmnoPQRstuvWXYZ`

#### 2. **GEMINI_API_KEY** - API ключ Google Gemini
- Перейдите на [Google AI Studio](https://aistudio.google.com/app/apikeys)
- Нажмите "Create API key"
- Скопируйте полученный ключ
- Пример: `AIzaSyD...` (длинная строка)

#### 3. **GEMINI_PROMPT** (опционально)
- Промт для настройки поведения AI при ответе на свободные вопросы
- По умолчанию: `"Ответь на вопрос: {user_message}"`
- Пример кастомного промта: `"Ты ассистент помощника студентов БНТУ. Помогай ответить на вопросы об общежитии: {user_message}"`


## 🚀 Запуск
